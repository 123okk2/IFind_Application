# IFind_Application
IFind 어플리케이션
(2019년 2학기 창의융합종합설계1 팀 과제)

#주제
기존의 단순 공고식 미아 찾기 시스템에서 벗어나,
사용자들(부모들)이 직접 참여하여 미아를 신고하고 제보하는 시스템.
